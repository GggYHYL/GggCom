# Input 输入框

通过鼠标或键盘输入字符

## 基础用法

:::demo 使用 `type`来定义输入框样式。
@examples/input/Basic.vue
:::

### Attributes

| 属性名      | 说明                   | 类型          |
| ----------- | ---------------------- | ------------- |
| type        | 用于控制该输入框的类型 | text,textarea |
| placeholder | 提示语                 | string        |
