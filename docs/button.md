# Button 按钮

常用的操作按钮。

## 基础用法

:::demo 使用 `type`来定义按钮样式。
@examples/button/Basic.vue
:::

### Attributes

| 属性名 | 说明                 | 类型                                   |
| ------ | -------------------- | -------------------------------------- |
| type   | 用于控制该按钮的类型 | primary,default,danger,success,warning |

### slots

| 插槽名  | 说明           |
| ------- | -------------- |
| default | 自定义按钮类型 |
